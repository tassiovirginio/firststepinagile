Seja bem vindo ao First Step in Agile

Aqui você vai aprender alguns fundamentos do gerenciamento Ágil focado em desenvolvimento de software utilizando o framework Scrum.
Além de ser um tutorial de alguns dos principios do Scrum, esse sistema também é uma ferramenta com todos os fluxos necessários para gerenciar projetos de forma completa. Vamos explicar de um modo geral cada passo que você deve seguir. no centro superior dessa página você pode encontrar um menu numerado, cada numero é o passo que deve ser seguido, e dentro de cada página se tem mais informações sobre o que esta acontecendo...
Vamos a uma listagem rápida:

0 - Start -> Essa página.

1 - Histórias -> Essa página se destina a elaboração das histórias do backlog

2 - Atividades -> Nessa página podemos gerar as atividades das histórias do backlog.

3 - Sprints -> Nessa pagina se faz a seleção das histórias para o sprint selecionado.

4 - Kanban -> E finalmente quando já estivermos com todas as nossas histórias com suas atividades poderemos iniciar os trabalhos e ver o progresso através do quadro ágil.

Seja bem vindo ao First Step In Agile e bom aprendizado.
