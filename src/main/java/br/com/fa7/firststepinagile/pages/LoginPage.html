First Step In Agile
Seja bem vindo ao First Step in Agile

Aqui você vai aprender alguns fundamentos do gerenciamento Ágil focado em desenvolvimento de software utilizando o framework Scrum.


Além de ser um tutorial de alguns dos principios do Scrum, esse sistema também é uma ferramenta com todos os fluxos necessários para gerenciar projetos de forma completa.

Vamos explicar de um modo geral cada passo que você deve seguir. após fazer o login você irá visualizar um menu no centro superior da página, cada numero é o passo que deve ser seguido, e dentro de cada página se tem mais informações sobre o que esta acontecendo...
Vamos a uma listagem rápida:

1 - Projetos -> Página responsavel em gerenciar os projetos.

2 - Backlog -> Essa página se destina a elaboração das histórias do backlog

3 - Atividades -> Nessa página podemos gerar as atividades das histórias do backlog.

4 - Sprints -> Nessa pagina se faz a criação e seleção do Sprint.

5 - Sprints Backlog -> Nessa pagina se faz a seleção das histórias para o sprint selecionado.

6 - Kanban -> E finalmente quando já estivermos com todas as nossas histórias com suas atividades poderemos iniciar os trabalhos e ver o progresso através do quadro ágil.

Seja bem vindo ao First Step In Agile e bom aprendizado.
