First Step In Agile


Aqui se inicia o tutorial para você aprender passo-a-passo os principios do desenvolvimento Ágil de Software, com foco em gestão dos projetos.

Vamos explicar de um modo geral cada passo que você deve seguir... no centro superio dessa página você pode encontrar um menu numerado, cada numero é o passo que deve ser seguido, e dentro de cada página se tem mais informações sobre o que esta acontecendo...
Vamos a uma listagem rapida:

0 - Start -> Essa página.

1 - Histórias -> Essa página se destina a elaboração das histórias do backlog

2 - Atividades -> Nessa página podemos gerar as atividades das histórias do backlog.

3 - Sprint -> Nessa pagina se faz a seleção das histórias para o sprint selecionado.

4 - Kanban -> E finalmente quando já estivermos com todas as nossas histórias com suas atividades poderemos iniciar os trabalhos e ver o progresso atravez do quadro ágil.

Seja bem vindo ao First Step In Agile e bom aprendizado.
