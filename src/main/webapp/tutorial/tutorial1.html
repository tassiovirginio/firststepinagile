First Step In Agile


Aqui se inicia o tutorial para você aprender passo-a-passo os principais principios do desenvolvimento Ágil de Software, focado na gestão dos projetos.

Vamos explicar de um modo geral cada passo que voce deve seguir... no centro superio dessa página você pode encontrar um menu numerado, cada numero é o passo e que deve ser seguido, e detro de cada pagina se tem mais informações sobre o que esta contecendo...
Vamos a uma lsitagem rapida:

1 - Start -> Página inicial onde se apresenta o sistema e como sera abordado os temas propostos.

2 - Estórias -> Essa página se destina a elaboração das estorias.

3 - Sprint -> Essa pagina a seleção das estorias para o sprint selecionado.

4 - Atividades -> O proximo passo despois de selecionar as estorias do sprint atual é gerar suas atividades.

5 - Kanbam -> E finalmente quando já estivermos com todas as nossas estorias com suas atividades poderemos iniciar os trabalhos e ver o progresso atravez do kanbam.

Seja bem vindo ao First Step In Agile e bom aprendizado.
