Essa é a página onde vamos criar os projetos, esse página foi elabora para melhorar a organização das historias, e para o sistema poder ser usado por mais de uma equipe.

Aqui você pode criar o Projeto e pode convidar outras pessoas para fazer parte do mesmo, atravéz do email do integrante da equipe.

Na caixa "Meus Projetos", serão exibidos todos os projetos que o usuário logado criou.

Na caixa "Projetos Convidados", serão exibidos todos os projetos que o usuário logado foi convidado a participar.

O usuario tem a opção de "selecionar" qualquer projeto que esta sendo exibido, para ser o projeto de trabalho atual. Após selecionado o nome do projeto irá aparecer na centro da barra superior do sistema.
